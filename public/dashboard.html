Hunting Tips

  • Inspect Elements: Right-click and use "Inspect" to view HTML
  • View Source: Press Ctrl+U (Cmd+Option+U on Mac) to see raw HTML
  • Console: Press F12 and check the Console tab for hidden messages
  • Network Tab: Monitor API requests and responses
  • Images: Check image alt text and metadata
  • CSS: Inspect styles for hidden elements
